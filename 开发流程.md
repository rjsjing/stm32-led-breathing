# STM32F103C8T6 LED 呼吸灯 (PWM) — 开发流程

## 1. 项目概述

- **MCU**: STM32F103C8T6（Cortex-M3, 64KB Flash, 20KB SRAM, 72MHz）
- **外设**: TIM2 CH1 PWM 输出 (PA0) + LED
- **通信方式**: PWM（1kHz, TIM2_CH1, PA0）
- **构建工具**: Keil MDK uVision5 / EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)
- **代码仓库**: [GitHub: rjsjing/stm32-pwm-led-breath](https://github.com/rjsjing/stm32-pwm-led-breath)

---

## 2. 技术栈

- **MCU**: STM32F103C8T6（Cortex-M3）
- **库**: ST 标准外设库 v3.5.0
- **构建工具**: Keil MDK uVision5 + EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)

---

## 3. 硬件连接

| STM32F103C8T6 | 连接目标 |
|---------------|----------|
| PA0           | LED（TIM2_CH1 PWM, 低电平点亮） |
| 3.3V          | VCC     |
| GND           | GND     |

## 5. 开发步骤

### 5.1 依赖

- Keil MDK v5 + STM32F10x 设备包
- VSCode + EIDE 插件 (`cl.eide`)
- 编译宏: `USE_STDPERIPH_DRIVER`, `STM32F10X_MD`

### 5.2 编译

```
Keil:  打开 Project.uvprojx → F7 编译
EIDE:  Ctrl+Shift+P → "EIDE: Build"
```

### 5.3 烧录

```
Keil:  F8 烧录（ST-Link SWD）
CLI:   STM32_Programmer_CLI.exe -c port=SWD -w build/Target\ 1/Project.hex
```

### 5.4 功能验证

- LED 渐亮渐暗循环（呼吸效果）
- PWM 占空比 0→999→0 循环变化
